## Scripts

Make sure you have nodejs and yarn installed. Install dependencies with:

```bash
yarn
```

Once it's done start up a local server with:

```bash
yarn start
```

To view the components and style guide:

```bash
yarn storybook
```

To run tests:

```bash
yarn test
```
